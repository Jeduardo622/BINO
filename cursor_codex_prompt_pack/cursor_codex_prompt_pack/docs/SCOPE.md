# Scope

## In Scope
- structured intake form
- curated-source retrieval
- deterministic filtering where fields exist
- ranked matches
- plain-English eligibility summary
- source citations on every surfaced result
- canonical source links
- visible source timestamps / last checked state
- direct contact path or practical next-step path where available
- optional outreach draft generation
- simple save/share
- lightweight internal review or QA surface
- privacy-safe data handling
- deletion/remove support if practical
- polished loading, empty, error, and success states
- migrations, tests, and run docs

## Primary sources for MVP
- ClinicalTrials.gov
- curated pediatric neuro / stroke / rehab program pages

## Out of Scope
- diagnosis recommendations
- treatment recommendations
- definitive eligibility determinations
- EMR or EHR integrations
- automated enrollment
- campaign workflows
- CRM workflows
- broad scraping infrastructure
- web-scale crawling
- clinician workflow suite
- broad condition expansion
- autonomous follow-up
- uncited AI matching
