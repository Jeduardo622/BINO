# Product

## Name
Pediatric Stroke Trial and Rehab Navigator

## Goal
Help parents go from a structured child profile to 3 to 5 credible, citation-backed trial or rehab options in under 10 minutes.

## Product type
A lean SaaS web app for discovery and navigation.

## This product IS
- a discovery tool
- a navigation tool
- a source-backed research aid
- a parent-facing workflow for finding possible next steps

## This product is NOT
- a medical decision tool
- a treatment recommendation engine
- a diagnosis tool
- a CRM
- an enrollment system
- a chatbot-first product

## Primary user
Parents of children ages 5 to 18 with:
- pediatric stroke
- hemiparesis post-stroke

## Validation question
Can a parent get to credible, reviewable next-step options faster than current manual search?
